# test_github
This repo was created to support my github workshops as a way to get you to start using github. Follow these steps to practice editing code collaboratively after [creating a github login](https://docs.github.com/en/get-started/start-your-journey/creating-an-account-on-github) and downloading [github desktop](https://desktop.github.com/download/).

## Setup
1. **Setup GitHub folder**: Somewhere within your computer's folders, create a folder titled "GitHub" which will be used to store all your working repos from here on out. Mine is found at: C:\Users\mmbet\Documents\GitHub
2. **Clone this repo onto your desktop**: Open github desktop and login > under "current repository," click add > clone repository > find this repo, "test_github" > click "clone." This repo will now show as your current repository.
3. **Fetch code**: Click "fetch origin" to refresh and make sure your repo is up to date.
